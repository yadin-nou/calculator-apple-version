# calculator-apple-version

Project Calculator Apple Version

- This project built by using technology Javascript, CSS and HTML

Project Workflow:

1. Create HTML
   - Using google font
   - Using element such <div> which has class .wrapper, inside has element <div> class calculator. Inside this <div> we have one <div> for display result and 19 <div> elements to create button for calculator.
2. Apply CSS
   - Using color which is similar to apple calculator in Mac.
   - Using flex box for alignment the calculator
   - Using grid box fro button
3. Create each button clicked template
4. Get the value the button and store it in the global variable
5. display the string from the global variable
6. Make the AC button work
7. Make "=" button work
8. Make the C button work
9. Avoid the last operator and show the result
10. Prevent multiple operations to press
11. Allow only one "." per number set
12. add button clicked effect
13. Bind the keypress on the calculator
